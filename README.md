# Calculator

A simple retro-style web calculator built with vanilla HTML, CSS, and JavaScript.

## Features

- Basic operations: addition, subtraction, multiplication, division
- Clear (`C`) button to reset calculator state
- Backspace (`Back`) button to undo the last input

## Keyboard Support

- Digits: `0-9`
- Operators: `+`, `-`, `*`, `/`
- Decimal: `.`
- Evaluate: `Enter` or `=`
- Backspace: `Backspace`
- Clear: `Escape`

## Project Structure

- `index.html` - calculator layout and buttons
- `style.css` - retro visual styling and layout
- `script.js` - calculator logic and keyboard handlers

## Run Locally

Open `index.html` in your browser.
